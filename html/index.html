ZeroWidth.Space

What is it?

A zerowidthspace is a space that only serves to break up a word when formatting

Example

This​is​an​example​of​a​very​long​word​created​with​zero​width​spaces.​If​you​resize​the​window/rotate​your​phone​to​see​how​it​always​line-breaks​on​a​word.

Non-Example

Thissimplyhasnospacesandasyoucanseeitistreatedasallonewordwhichjustkeepsgoingandgoingandgoing

Why should I avoid it?

If you're working on a website there's a good chance that someone will want to copy/paste it, automate using it, etc. Zero-width spaces will create no end of hassle to those people.
